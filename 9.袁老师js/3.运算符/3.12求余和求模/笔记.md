## 求余和求模

% : 求余

x % y

求余 x rem y: x - n * y, n为商取整（向0取整）

```js
x = 7,y = 3, x rem y = 7 - 2*3 = 1

n = x / y = 2.33333~ 2
```

```js
x = 7, y = -3, x rem y =  7 - n*y = 1
n = 7 / -3 ~ -2

```

```js
x = -7, y = 3, x rem y =  -7 - n*y = -1
n = -7 / 3 ~ -2

```

**余数和被除数的符号相同**

2. 求模 x mod y : x - n * y, n为余数取整（向下取整）

**求模的符号除数的符号相同**
